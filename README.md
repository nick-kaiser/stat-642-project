## Setup

This project uses [`renv`](https://rstudio.github.io/renv/) to manage R package dependencies. To restore the environment:

1. Install R (if not already installed).
2. Open the project in RStudio or set your working directory to the project root.
3. Install `renv` if you don't have it: `install.packages("renv")`
4. Restore the project library: `renv::restore()`

This will install all required packages at the versions recorded in `renv.lock`.

## PowerPoint Presentation Guidelines (copied from project rubric)
- Include the title and your team number clearly (UINs are not needed).
- Begin with a clear introduction and data description, including the data source. If you designed the experiment, explain the design clearly.
- Clearly specify the model, describing all parameters, variables, and subscripts, and state all assumptions. If assumptions are violated, explain how you addressed them to ensure a sound model.
- Explain what is interesting about the data and clearly state your main research questions.
- Clearly communicate the analysis and results, including checks of all assumptions.
- When the null hypothesis is rejected and differences are found, clearly indicate which groups differ or how variables increase or decrease.
- Include clearly labeled cut-and-paste output where appropriate.
- Your presentation should contain all necessary details.
- Save your PowerPoint file as a PDF for submission.
- Provide the program file(s) along with the data.
- Your starting model must include at least three predictors. Interaction terms do not count toward the predictor total.
- Make your models and analyses as interesting and meaningful as possible.
- Include detailed results and a clear conclusion.
